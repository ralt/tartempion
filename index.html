Tartempion

The web framework for people who like to eat some pie.

View the Project on GitHub Ralt/tartempion

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Why should you use tartempion?

  • You want to reuse parts of your code across different projects.
  • You want to inject dependencies without thinking about it.
  • You want express but you feel like it's too slow to set up a new project.
  • You want to have well-organized projects.

Then tartempion is for you!

Installation

npm install tartempion -g

Example application

There is a blog using tartempion. The repository is available there: tartempion-blog.
